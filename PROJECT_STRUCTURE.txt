File Descriptions
====================================================

CORE PYTHON FILES:
------------------

1. state.py
   - PDFDocument: Input document dataclass
   - DocumentStructure: Extracted PDF structure
   - Guideline: Single extracted guideline
   - EnrichedGuideline: Guideline with metadata
   - GraphState: Main workflow state (TypedDict)
   - DocumentBatchState: Parallel processing state
   - ProcessingError: Error tracking

2. config.py
   - ModelConfig: Gemini & OpenAI configuration
   - QdrantConfig: Vector DB configuration
   - ProcessingConfig: Workflow parameters
   - SystemConfig: Master configuration
   - Category mapping and priority rules

3. nodes.py
   - document_analyzer_node: PDF analysis with Gemini 2.0
   - content_extractor_node: Guideline extraction & classification
   - deduplication_node: Semantic similarity deduplication
   - collection_router_node: Route to Qdrant collections
   - metadata_enricher_node: Add priority, complexity
   - vector_storage_node: Store in Qdrant with embeddings

4. graph.py
   - create_workflow(): Main workflow builder
   - Conditional edges for flow control
   - Export as 'app' for deployment
   - Visualization helper
   - Example usage

5. parallel_processor.py
   - create_parallel_workflow(): Send() API workflow
   - parallel_document_processor(): Process single doc
   - MemoryAwareBatchProcessor: Batch with memory limits
   - Dispatcher and aggregator nodes

6. error_handling.py
   - retry_with_backoff: Exponential backoff decorator
   - CircuitBreaker: Circuit breaker pattern
   - ErrorRecoveryStrategy: Error classification
   - MemoryMonitor: Memory tracking
   - FaultTolerantBatchProcessor: Batch error tolerance

7. api.py
   - FastAPI application
   - POST /upload: Single PDF upload
   - POST /upload-batch: Batch PDF upload
   - GET /batch/{id}: Status tracking
   - POST /search: Semantic search
   - GET /collections: List collections
   - GET /guideline/{id}: Get specific guideline
   - Background task processing

8. __init__.py
   - Package initialization
   - Exports: app, GraphState, config, etc.
